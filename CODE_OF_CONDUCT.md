# Code of Conduct

The CAP project adopts the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)
as its Code of Conduct.

Read the full text:
<https://www.contributor-covenant.org/version/2/1/code_of_conduct/code_of_conduct.md>

## Reporting

To report unacceptable behavior, contact the project maintainers via:

- GitHub Private Vulnerability Reporting on any `cap-protocol/*` repo (works
  for conduct issues as well as security)
- `conduct@cap-protocol.example` *(placeholder — will be set once the org
  email is provisioned)*

Reports are handled confidentially. The maintainers will investigate and
respond within 14 days. Enforcement follows the Contributor Covenant
enforcement ladder (correction → warning → temporary ban → permanent ban).
